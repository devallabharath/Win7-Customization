How to install Boot Screen

1.Download Windows 7 Boot Updater from this address:
http://www.coderforlife.com/projects/win7boot/#Download

2.Run the "Win7BootUpdater.exe" file.

3.Click on "File" from menu and then "Load boot skin...".

4.Choose "Boot Screen.bs7" file from folder "Windows 10 Extras\Boot Screen".

5.Press "Apply" button.

All you are doing, doing at your own risk. For the possible consequences i am not responsible.

Tested and working on Windows 7 Service Pack 1 EN-US 32-bit and 64-bit.
