How to install Fonts

1.Backup your Segoe UI font files from folder "C:\Windows\Fonts".

2.Copy all font files from folder "Windows 10 Extras\Fonts\Segoe UI" to "C:\Windows\Fonts" folder.

Or

Go to "Windows 10 Extras\Fonts\Segoe UI" folder, select all font files, right-click on the one selected and choose "Install" option.

3.Log off or reboot your computer.
