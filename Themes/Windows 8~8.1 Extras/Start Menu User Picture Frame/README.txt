How to change Start Menu User Picture Frame

1.Download Windows 7 User Picture Frame Changer from this address:
http://www.door2windows.com/windows-7-user-picture-frame-changer-change-your-user-picture-frame-with-one-click/
and follow instructions from this site how to change start menu user picture frame.

2.Choose start menu user picture frame size you want to use.

For example:

-Default DPI Display Size

In this case, select "Start Menu User Picture Frame" file from folder "Windows 8~8.1 Extras\Start Menu User Picture Frame\Default DPI Display Size".
