How to change Start Menu Button

1.Download Windows 7 Start Orb Changer from this address:
http://www.door2windows.com/windows-7-start-button-changer-change-your-windows-7-start-button-with-one-click/
and follow instructions from this site how to change start menu button.

2.Choose start menu button with features you want to use.

For example:

-Default DPI Dispaly Size
-Default Taskbar

In this case, select "Start Menu Button" file from folder "Windows 8~8.1 Extras\Start Menu Button\Default DPI Display Size\Default Taskbar".
