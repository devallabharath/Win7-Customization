How to get White Task Pane

It's recommended to install Windows 7 Service Pack 1 before you change system files.

How to take ownership (permission) of a file you can find on this address:
http://www.askvg.com/guide-how-to-take-ownership-permission-of-a-file-or-folder-manually-in-windows/
http://www.askvg.com/add-take-ownership-option-in-file-folder-context-menu-in-windows-vista/

FOR WINDOWS 7 32-BIT USERS

1.Take ownership (permission) of a file "shell32.dll" from folder "C:\Windows\System32" and rename it to, for example, "shell32_old.dll".

2.Copy "shell32.dll" file from folder "Windows 8~8.1 Extras\White Task Pane\32-BIT Users\System32" to "C:\Windows\System32" folder.

3.Reboot your computer.

FOR WINDOWS 7 64-BIT USERS

1.Take ownership (permission) of a file "shell32.dll" from folders "C:\Windows\System32" and "C:\Windows\SysWOW64" and rename it to, for example, "shell32_old.dll".

2.Copy "shell32.dll" file from folder "Windows 8~8.1 Extras\White Task Pane\64-BIT Users\System32" to "C:\Windows\System32" folder. 

Also copy "shell32.dll" file from folder "Windows 8~8.1 Extras\White Task Pane\64-BIT Users\SysWOW64" to "C:\Windows\SysWOW64" folder.

3.Reboot your computer.

All you are doing, doing at your own risk. For the possible consequences i am not responsible.

Tested and working on Windows 7 Service Pack 1 32-bit and 64-bit.
